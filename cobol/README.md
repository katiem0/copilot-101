# Copilot + Mainframe

Basic COBOL and JCL examples for exploring how GitHub Copilot helps
developers work with mainframe assets.

## Files
- [CUSTPROC.cbl](CUSTPROC.cbl) — COBOL program that reads a customer file and writes a formatted report, tracking read / valid / error counts.
- [CUSTPROC.jcl](CUSTPROC.jcl) — JCL job that runs `CUSTPROC` against a customer dataset and writes the report to SYSOUT.
