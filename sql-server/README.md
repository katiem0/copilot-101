# Copilot + SQL Server

Basic T-SQL examples for exploring how GitHub Copilot helps with SQL Server
development.

## Files
- [schema.sql](schema.sql) — small order-management schema (Customers, Products, Orders, OrderItems).
- [queries.sql](queries.sql) — sample queries, including one intentionally naive query useful for optimization exercises.
- [sp_GetCustomerOrders.sql](sp_GetCustomerOrders.sql) — stored procedure that builds dynamic SQL from a parameter (contains a deliberate SQL injection risk for review exercises).
